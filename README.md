# Multi-Mode SPI Controller (Verilog)

This project implements a Serial Peripheral Interface (SPI) controller in Verilog HDL.

## Features
- Supports SPI communication
- Configurable clock using clock divider
- Modular RTL design
- Verilog testbench for simulation

## Files
spi.v – SPI protocol implementation  
clk_divider.v – Clock divider module  
top_module.v – Top level integration  
spi_tb.v – Testbench for verification  

## Tools
Vivado Simulator
